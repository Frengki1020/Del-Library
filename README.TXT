========== ==========
Description:
Aplikasi LibraryNG adalah aplikasi berbasis web yang dapat digunakan untuk membantu/mendukung kegiatan Di bagian Perpustakaan Institut Teknologi Del, Aplikasi yang sudah di bangun ini akan memberikan manfaat yang sangat besar sekali kepada bagian staff perpustakaan dan juga mahasiswa Institut Teknologi Del . 
========== ==========
Features:
1. Authentication (login/logout mechanism),
2. three-roles authorization (member, staff & admin),
3. Menampilkan gambar dari setiap buku (atau eksemplar)
4. Member
    - Melihat daftar & detil buku 
    - Melihat daftar & detil eksemplar (copy)
    - Request peminjaman eksemplar
    - Cancel peminjaman jika belum di-approve)
    - Update credential (password) & profil
5. Staf 
    - Melihat daftar member & detil profilnya.
    - Melihat daftar & detil request peminjaman.
    - Melakukan approval terhadap request.
    - Melakukan pengembalian eksemplar.
    - Melihat daftar peminjaman yang belum jatuh tempo.
    - Melihat daftar peminjaman yang telah jatuh tempo (termasuk denda) ASC by due date.
    - Page untuk approve & return (single page).
6. Administrator
    - Assign & revoke role
 7. Lainnya
    - Sebuah eksemplar hanya dapat dipinjam satu kali dalam satu waktu.
    - RBAC
    - Menggunakan komponen yang didesain sendiri atau third party.
    - Menampilkan gambar dari setiap buku (atau eksemplar)
    - Menggunakan web service untuk mencari buku di perpustakaan lain.
    - Desain yang intuitif.
8. Penambahan Extention
    - mbMenu
    - clEditor
========== ==========
Minimum software requirements:
1. PHP-ready web server (Apache 2.4.x or higher),
2. MySQL database (version 5.5.x or higher),
3. PHP engine (version 5.5 or higher),
========== ==========
Installation:
1. Pastikan web server & database server terinstal dengan baik dan dapat digunakan,
2. Buat user pada MySQL dengan credential user/user,
3. Impor SQL script yang disediakan,
4. letakan NetBeans project ke direktori C:\xampp\htdocs dan pada directori ini harus sudah terinstal sebelumnya Yii framewok.
